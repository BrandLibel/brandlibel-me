#BrandLibel.me
My personal website built using React.js and Node.js

##Requirements
- [node.js](https://nodejs.org) (I used version 10.16.0)
- [yarn](https://yarnpkg.com/)

##Install:
- `yarn install`

##Test locally:
- `yarn start`
- visit `localhost:8080`

##Build and run in production:
- `yarn build`
- `yarn prod`
- visit `localhost:3000`
